## 工程管理

### 创建新工程

1. 点击 **文件** → **新建工程** 或按 `Ctrl+N`
2. 在对话框中输入：
   - **工程名称**：输入有意义的名称
   - **工程路径**：选择保存位置
   - **工程描述**：（可选）添加描述
3. 点击 **确定**

工程结构：
```
MyProject/
├── project.json          # 工程配置文件
├── Document/             # 文档目录
├── Backup/               # 备份目录
├── src/                  # 源代码目录
└── data/                 # 数据目录
    ├── raw/              # 原始数据
    └── processed/        # 处理后数据
```

### 打开工程

1. 点击 **文件** → **打开工程** 或按 `Ctrl+O`
2. 选择工程文件夹
3. 点击 **确定**

### 保存工程

- 点击 **文件** → **保存工程** 或按 `Ctrl+S`
- 工程配置会自动保存

## 数据导入

### 支持的文件格式

- **.ns3**：连续宽频信号（LFP和Spike）
- **.nev**：Spike事件和波形数据
- **.mbm**：行为任务参数和事件

### 导入单个试验

1. 点击 **数据加载** 标签页
2. 点击 **导入试验** 按钮
3. 在对话框中：
   - 选择 **NS3文件**（必需）
   - NEV和MBM文件会自动检测
   - 输入 **试验名称**
   - 添加 **试验描述**（可选）
4. 点击 **导入**

### 批量导入

1. 点击 **批量导入** 按钮
2. 选择包含数据文件的文件夹
3. 系统会自动识别配套的试验文件
4. 选择要导入的试验
5. 点击 **确认导入**

### 导入进度

导入过程中会显示进度条：
- 解析NS3文件
- 解析NEV文件
- 创建HDF5文件
- 添加Spike数据
- 解析行为数据

## 数据分析

### 选择数据

1. 在左侧导航栏点击试验名称
2. 在参数面板的 **数据选择** 区域：
   - 勾选要分析的数据项
   - 支持多选

### 配置时间对齐

当选择多个数据项时：
1. 点击 **时间对齐** 按钮
2. 在对话框中设置：
   - **对齐起始时间**：分析开始时间
   - **持续时间**：分析时长
3. 或使用 **建议对齐** 自动计算
4. 点击 **确认对齐**

### Spike检测

1. 点击 **Spike检测** 标签页
2. 选择 **阈值检测**
3. 配置参数：
   - **阈值系数**：相对于噪声标准差的倍数（默认4.5）
   - **阈值类型**：positive/negative/both
   - **检测窗口**：波形提取窗口大小
   - **不应期**：两次检测的最小间隔
   - **滤波器**：高通/低通截止频率
4. 选择要分析的通道
5. 点击 **运行检测**

### Spike排序

1. 点击 **Spike排序** 标签页
2. 选择排序方法：
   - **PCA+KMeans**：主成分分析+K均值聚类
   - **GMM**：高斯混合模型
3. 配置参数：
   - **主成分数量**：PCA降维维度
   - **聚类数量**：期望的神经元数量
4. 点击 **运行排序**

### LFP分析

#### 功率谱分析
1. 点击 **LFP分析** 标签页
2. 选择 **功率谱**
3. 配置参数：
   - **频率范围**：分析频率范围
   - **窗口大小**：Welch方法窗口
   - **重叠率**：窗口重叠比例
4. 点击 **运行分析**

#### 时频分析
1. 选择 **时频图**
2. 配置参数：
   - **窗口大小**：STFT窗口
   - **重叠样本**：样本重叠数
3. 点击 **运行分析**

### 行为分析

#### PSTH分析
1. 点击 **行为分析** 标签页
2. 选择 **PSTH**
3. 配置参数：
   - **时间窗口**：事件前后时间范围
   - **分箱大小**：时间分箱大小
   - **事件类型**：对齐的事件类型
4. 点击 **运行分析**

#### 栅格图
1. 选择 **栅格图**
2. 配置参数：
   - **时间窗口**：显示时间范围
   - **试次排序**：排序方式
3. 点击 **运行分析**

### 解码分析

1. 点击 **解码** 标签页
2. 选择解码方法：
   - **LDA**：线性判别分析
   - **SVM**：支持向量机
   - **随机森林**：随机森林分类器
3. 配置参数：
   - **交叉验证折数**：K折交叉验证
   - **测试集比例**：训练/测试分割比例
4. 点击 **运行解码**

## 可视化

### 图表类型

- **信号波形**：原始LFP信号
- **Spike波形**：检测到的Spike波形
- **栅格图**：Spike时间栅格
- **PSTH**：刺激后时间直方图
- **调谐曲线**：刺激-响应关系
- **功率谱**：频谱功率分布
- **时频图**：时间-频率能量分布

### 图表交互

- **缩放**：鼠标滚轮或工具栏按钮
- **平移**：拖拽或工具栏按钮
- **选取**：框选区域
- **重置**：恢复默认视图

### 多标签页管理

- 每个分析结果在新标签页显示
- 点击标签切换
- 点击 × 关闭标签页
- 支持同时显示多个图表

## 结果导出

### 导出图表

1. 在图表工具栏点击 **保存** 按钮
2. 选择格式：
   - **PNG**：便携式网络图形
   - **PDF**：便携式文档格式
   - **SVG**：可缩放矢量图形
3. 选择保存路径
4. 点击 **保存**

### 导出数据

1. 在数据项上右键点击
2. 选择 **导出**
3. 选择格式：
   - **MAT**：MATLAB格式
   - **NumPy**：.npy格式
   - **CSV**：逗号分隔值
4. 选择保存路径
5. 点击 **保存**

## 常见问题

### Q1: 导入数据时提示文件不存在？

**A**: 请检查：
1. 文件路径是否正确
2. 文件是否被其他程序占用
3. 文件权限是否足够

### Q2: 分析结果为空？

**A**: 可能原因：
1. 参数设置不合理（如阈值过高）
2. 数据质量问题
3. 时间对齐设置错误

### Q3: 程序运行缓慢？

**A**: 优化建议：
1. 减少同时分析的数据量
2. 降低采样率（如适用）
3. 关闭不必要的图表
4. 增加系统内存

### Q4: 如何批量处理多个试验？

**A**: 
1. 使用 **批量导入** 功能导入所有试验
2. 编写脚本调用算法API（高级用户）

### Q5: 如何添加自定义算法？

**A**: 参考开发指南：
1. 继承 `BaseAlgorithm` 基类
2. 实现 `run` 方法
3. 定义参数模式
4. 注册到算法调度器

### Q6: HDF5文件格式是什么？

**A**: HDF5是一种高效的科学数据存储格式：
- 支持大文件（>2GB）
- 支持数据压缩
- 支持分块读写
- 跨平台兼容

### Q7: 如何备份工程？

**A**: 
1. 手动复制工程文件夹
2. 或使用 **工具** → **备份工程** 功能

### Q8: 采样率对分析有什么影响？

**A**: 
- **Spike检测**：需要高采样率（>20kHz）
- **LFP分析**：通常2kHz足够
- **行为分析**：与神经数据对齐即可
