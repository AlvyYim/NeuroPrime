## 1. 原始数据格式说明

### 1.1 文件类型总览

| 文件扩展名 | 文件格式 | 数据类型 | 用途说明 | 是否必须 |
|-----------|---------|---------|---------|---------|
| **.ns3** | Blackrock NSx | 连续电信号 | 宽频信号（包含LFP和Spike） | ✅ 必须 |
| **.nev** | Blackrock NEV | Spike事件 | 神经脉冲时间戳和波形 | ✅ 必须 |
| **.mbm** | 自定义二进制 | 行为参数 | 实验任务参数和刺激条件 | ⚪ 可选 |
| **.ccf** | 配置文件 | 电极映射 | 电极编号和位置映射 | ⚪ 可选 |
| **.eye** | 眼动数据 | 眼动追踪 | 注视点位置（暂不导入） | ❌ 暂不 |
| **.log** | 文本日志 | 实验日志 | 记录实验过程信息 | ⚪ 可选 |

### 1.2 .ns3 文件详细结构

#### 文件头（Basic Header）- 314字节

```c
struct NS3BasicHeader {
    char FileType[8];           // "NEURALSG" 或 "NEURALCD"
    char Version[2];            // 版本号 [major, minor]
    uint32 HeaderSize;          // 头文件总大小
    char Label[16];             // 记录标签
    char Reserved[256];         // 保留空间
    uint32 Period;              // 采样周期
    uint32 ClockFs;             // 时钟频率 (30000 Hz)
    uint16 TimeOrigin[8];       // 时间原点 (年月日时分秒毫秒)
    uint32 NumElec;             // 电极数量
};
```

#### 扩展头（Extended Header）- 每个电极66字节

```c
struct NS3ExtendedHeader {
    char PacketID[2];           // "CC" = 连续通道
    uint16 ElecID;              // 电极ID
    char ElecLabel[16];         // 电极标签
    char Connector;             // 连接器编号
    char Pin;                   // 引脚编号
    int16 MinDigitalValue;      // 最小数字值
    int16 MaxDigitalValue;      // 最大数字值
    int16 MinAnalogValue;       // 最小模拟值 (uV)
    int16 MaxAnalogValue;       // 最大模拟值 (uV)
    char AnalogUnit[16];        // 单位 "uv"
    uint32 HighCutFreq;         // 高通截止频率
    uint32 HighCutOrder;        // 高通阶数
    uint16 HighCutType;         // 高通类型
    uint32 LowCutFreq;          // 低通截止频率
    uint32 LowCutOrder;         // 低通阶数
    uint16 LowCutType;          // 低通类型
};
```

#### 数据包（Data Packet）

```c
struct NS3DataPacket {
    uint8 Header;               // 0x01 数据包头标记
    uint32 Timestamp;           // 时间戳（时钟周期数）
    uint32 NumSamples;          // 样本数
    int16 Data[NumElec][NumSamples];  // 实际数据 [电极数×样本数]
};
```

**关键参数**:
- 采样率: 30,000 Hz
- 数据类型: int16
- 单位转换: `物理值(uV) = 数字值 × (MaxAnalogValue - MinAnalogValue) / (MaxDigitalValue - MinDigitalValue)`

### 1.3 .nev 文件详细结构

#### 文件头（Basic Header）- 336字节

```c
struct NEVBasicHeader {
    char FileType[8];           // "NEURALEV"
    char Version[2];            // 版本号
    char ExtraFlags[16];        // 波形标志
    uint32 HeaderSize;          // 头文件大小
    uint32 PacketSize;          // 数据包大小 (通常为108字节)
    uint32 ClockFs;             // 时钟频率
    uint32 WaveformFs;          // 波形采样率
    uint16 TimeOrigin[8];       // 时间原点
    char SoftwareVersion[32];   // 软件版本
    uint32 NumExtend;           // 扩展头数量
};
```

#### 扩展头类型

| PacketID | 类型 | 说明 |
|---------|------|------|
| "NEUEVWAV" | 神经事件波形 | 包含电极阈值、波形长度等 |
| "NEUEVLBL" | 神经事件标签 | 电极标签信息 |
| "NEUEVFLT" | 神经事件滤波 | 滤波器参数 |
| "DIGLABEL" | 数字标签 | 数字输入配置 |
| "NSASEXEV" | 实验信息通道 | 实验事件配置 |

#### 数据包（Data Packet）- 108字节

```c
struct NEVDataPacket {
    uint32 Timestamp;           // 时间戳（时钟周期数）
    uint16 ElecID;              // 电极ID
    uint8 Unit;                 // 聚类单元 (0=未排序, 1-5=已排序)
    uint8 Reserved;             // 保留
    uint16 DigInput;            // 数字输入（事件标记）
    int16 Waveform[48];         // 波形数据 (48个样本)
};
```

**关键事件标记**（DigInput字段）:

| 值 | 事件类型 | 说明 |
|----|---------|------|
| 1 | CBEVENT_START | 实验开始 |
| 2 | CBEVENT_STOP | 实验结束 |
| 3 | CBEVENT_PAUSEON | 试次结束 |
| 4 | CBEVENT_PAUSEOFF | 试次开始 |
| 5 | CBEVENT_ABORT | 试次中止 |
| 128 | CBEVENT_VSGTRIG | VSG触发 |
| 255/254/253 | CBEVENT_STIMCND | 刺激条件 |

### 1.4 .mbm 文件结构

.mbm文件是自定义二进制格式，包含行为任务参数：

```c
struct MBMData {
    int32 StimID;               // 刺激ID
    int32 RespCode;             // 反应代码 (>0=有效试次)
    // 其他实验特定参数...
};
```

```

│   ├── unit_info                # 单元信息 (Compound Type)
│   │   ├── unit_id              # 单元ID (uint16)
│   │   ├── channel_id           # 通道ID (uint16)

└── /behavior                    # 行为数据组
    ├── trial_info               # 试次信息 (Compound Type)
    │   ├── trial_id             # 试次ID (uint16)
    │   ├── start_time           # 开始时间 (float32)
    │   ├── end_time             # 结束时间 (float32)
    │   ├── stimulus_id          # 刺激ID (int32)
    │   ├── stimulus_type        # 刺激类型 (字符串)
    │   ├── response_code        # 反应代码 (int32)
    │   └── valid_trial          # 是否有效 (bool)
    ├── events                   # 事件信息 (Compound Type)
    │   ├── event_time           # 事件时间 (float32)
    │   ├── event_type           # 事件类型 (字符串)
    │   └── event_value          # 事件值 (int32)
    └── mbm_data                 # MBM原始数据 (可选)
```

### 2.3 数据类型详细说明

#### /metadata 组

| 数据集 | 数据类型 | 维度 | 说明 |
|-------|---------|------|------|
| experiment_name | string | scalar | 实验名称，如"猕猴V1区电生理实验" |
| trial_name | string | scalar | 试验名称，如"FC_Grating_014" |
| import_time | string | scalar | 导入时间戳，ISO格式 |
| source_files | string[] | 1D | 源文件列表，如["FC_Grating_014.ns3", "FC_Grating_014.nev"] |
| sampling_rate | float32 | scalar | 原始采样率，30000.0 Hz |
| num_channels | int32 | scalar | 通道数量，如96 |
| duration | float32 | scalar | 记录总时长，单位秒 |
| time_alignment | float32 | scalar | 时间对齐基准点（第一个TTL时间） |
| conversion_factor | float32 | scalar | 数字值到uV的转换因子 |

#### /signals 组

| 数据集 | 数据类型 | 维度 | 说明 |
|-------|---------|------|------|
| raw_data | float32 | [C × S] | 原始宽频信号，C=通道数，S=样本数，单位uV |
| lfp_data | float32 | [C × L] | LFP信号，降采样到1000Hz，单位uV |
| sampling_rate_lfp | float32 | scalar | LFP采样率，1000.0 Hz |
| channel_info | Compound | [C] | 通道信息表格 |

**channel_info Compound Type**:
```
{
    channel_id: uint16,
    electrode_id: uint16,
    electrode_label: string[16],
    connector: string[2],
    pin: uint8,
    unit: string[8]
}
```

#### /spikes 组

| 数据集 | 数据类型 | 维度 | 说明 |
|-------|---------|------|------|
| spike_times | float32 | [N] | Spike时间戳，相对于试次开始，单位秒 |
| spike_times_abs | float32 | [N] | 绝对时间戳，相对于记录开始，单位秒 |
| spike_waveforms | float32 | [N × 48] | Spike波形，48个样本，单位uV |
| cluster_labels | uint8 | [N] | 聚类标签，0=未排序，1-5=已排序单元 |
| channel_ids | uint16 | [N] | Spike所在通道ID |
| unit_info | Compound | [U] | 单元信息表格，U=单元数 |
| trial_indices | Reference | [T] | 每个试次的Spike索引，T=试次数 |

**unit_info Compound Type**:
```
{
    unit_id: uint16,
    channel_id: uint16,
    num_spikes: uint32,
    quality_score: float32
}
```

#### /behavior 组

| 数据集 | 数据类型 | 维度 | 说明 |
|-------|---------|------|------|
| trial_info | Compound | [T] | 试次信息表格，T=试次数 |

### 3.3 时间对齐配置接口

```python
class TimeAlignmentConfig:
    """
    时间对齐配置 - 在可视化/分析前对多组数据进行时间对齐
    
    使用场景：
    1. 用户在数据选择列表勾选多个数据项
    2. 点击"时间对齐"按钮，弹出时间对齐对话框
    3. 系统显示各数据的原始时间范围
    4. 用户设置统一的初始时间和持续时间
    5. 对齐参数传递给后续的可视化或分析算法
    """
    
    def __init__(self):
        # 各数据项的对齐参数
        self.data_configs: Dict[str, Dict] = {}  # {data_id: {'start': float, 'duration': float}}
        
        # 全局对齐参数（所有数据共用）
        self.global_start: float = 0.0      # 统一的初始时间（秒）
        self.global_duration: float = 10.0  # 统一的持续时间（秒）
        
        # 对齐模式
        self.sync_mode: str = 'global'  # 'global'(全局同步) / 'individual'(独立设置)
    
    def set_global_time_window(self, start: float, duration: float):
        """
        设置全局时间窗口（修改一行，其他行自动同步）
        
        Args:
            start: 初始时间（秒）
            duration: 持续时间（秒）
        """
        self.global_start = start
        self.global_duration = duration
        
        # 同步到所有数据项
        for data_id in self.data_configs:
            self.data_configs[data_id]['start'] = start
            self.data_configs[data_id]['duration'] = duration
    
    def add_data_item(self, data_id: str, original_time_range: Tuple[float, float]):
        """
        添加数据项到对齐配置
        
        Args:
            data_id: 数据项标识符
            original_time_range: 原始时间范围 (start, end)
        """
        self.data_configs[data_id] = {
            'original_range': original_time_range,
            'start': self.global_start,
            'duration': self.global_duration
        }
    
    def get_aligned_time_window(self, data_id: str) -> Tuple[float, float]:
        """
        获取指定数据项对齐后的时间窗口
        
        Returns:
            (对齐后起始时间, 对齐后结束时间)
        """
        config = self.data_configs.get(data_id, {})
        start = config.get('start', 0.0)
        duration = config.get('duration', 10.0)
        return (start, start + duration)
    
    def validate_alignment(self) -> Tuple[bool, str]:
        """
        验证对齐参数是否有效
        
        Returns:
            (是否有效, 错误信息)
        """
        for data_id, config in self.data_configs.items():
            orig_start, orig_end = config['original_range']
            aligned_start, aligned_end = self.get_aligned_time_window(data_id)
            
            # 检查对齐窗口是否在原始范围内
            if aligned_start < orig_start or aligned_end > orig_end:
                return False, f"数据项 {data_id} 的对齐时间范围超出原始范围"
        
        return True, ""
    
    def to_dict(self) -> Dict:
        """转换为字典，便于传递给算法"""
        return {
            'global_start': self.global_start,
            'global_duration': self.global_duration,
            'sync_mode': self.sync_mode,
            'data_configs': self.data_configs
        }
```

```python
from abc import ABC, abstractmethod
from typing import Dict, Any, Optional
import numpy as np


class AlgorithmInput:
    """算法输入数据标准格式"""
    
    def __init__(self):
        # 信号数据
        self.signals: Dict[str, np.ndarray] = {}      # 信号数据字典
        self.sampling_rate: float = 30000.0           # 采样率
        self.channel_ids: List[int] = []              # 通道ID列表
        
        # Spike数据
        self.spike_times: Optional[np.ndarray] = None # Spike时间戳
        self.spike_waveforms: Optional[np.ndarray] = None  # Spike波形
        self.cluster_labels: Optional[np.ndarray] = None   # 聚类标签
        
        # 行为数据
        self.trial_info: Optional[pd.DataFrame] = None # 试次信息
        self.events: Optional[pd.DataFrame] = None     # 事件信息
        
        # 元数据
        self.metadata: Dict[str, Any] = {}
        
        # 用户参数
        self.parameters: Dict[str, Any] = {}


class AlgorithmOutput:
    """算法输出数据标准格式"""
    
    def __init__(self):
        self.success: bool = False           # 执行状态
        self.message: str = ""               # 状态信息
        self.results: Dict[str, Any] = {}    # 分析结果
        self.figures: List[Dict] = []        # 图表配置列表
        self.export_data: Dict[str, Any] = {}  # 可导出数据


class BaseAlgorithm(ABC):
    """算法基类 - 所有分析算法必须继承"""
    
    # 算法元信息
    name: str = ""                       # 算法名称
    description: str = ""                # 算法描述
    category: str = ""                   # 算法类别
    version: str = "1.0"                 # 算法版本
    
    # 参数定义
    parameters_schema: Dict[str, Dict] = {}  # 参数模式定义
    
    def __init__(self):
        self.input_data: Optional[AlgorithmInput] = None
        self.output_data: Optional[AlgorithmOutput] = None
    
    @abstractmethod
    def validate_input(self, input_data: AlgorithmInput) -> Tuple[bool, str]:
        """
        验证输入数据
        
        Returns:
            (是否有效, 错误信息)
        """
        pass
    
    @abstractmethod
    def run(self, input_data: AlgorithmInput) -> AlgorithmOutput:
        """
        执行算法
        
        Args:
            input_data: 算法输入数据
            
        Returns:
            算法输出结果
        """
        pass
    
    def get_default_parameters(self) -> Dict[str, Any]:
        """获取默认参数"""
        defaults = {}
        for param_name, param_def in self.parameters_schema.items():
            defaults[param_name] = param_def.get('default')
        return defaults
    
    def get_parameter_schema(self) -> Dict[str, Dict]:
        """
        获取参数模式定义
        
        Returns:
            {
                'param_name': {
                    'type': 'float'/'int'/'string'/'bool'/'enum',
                    'default': default_value,
                    'range': [min, max],           # 数值类型
                    'options': ['opt1', 'opt2'],   # enum类型
                    'description': '参数描述'
                }
            }
        """
        return self.parameters_schema
```

### 4.3 算法调度器

```python
class AlgorithmScheduler:
    """算法调度器 - 管理所有算法的注册和执行"""
    
    def __init__(self):
        self._algorithms: Dict[str, Type[BaseAlgorithm]] = {}  # 内置算法
        self._custom_algorithms: Dict[str, Type[BaseAlgorithm]] = {}  # 自定义算法
        self._execution_history: List[Dict] = []  # 执行历史
    
    def register_builtin_algorithm(self, algorithm_class: Type[BaseAlgorithm]):
        """注册内置算法"""
        self._algorithms[algorithm_class.name] = algorithm_class
    
    def register_custom_algorithm(self, name: str, 
                                  algorithm_class: Type[BaseAlgorithm]) -> bool:
        """注册自定义算法"""
        # 验证算法接口
        if not issubclass(algorithm_class, BaseAlgorithm):
            return False
        
        # 测试算法
        test_result = self._test_algorithm(algorithm_class)
        if not test_result['valid']:
            return False
        
        self._custom_algorithms[name] = algorithm_class
        return True
    
    def get_algorithm_list(self) -> List[Dict]:
        """获取所有可用算法列表"""
        algorithms = []
        
        # 内置算法
        for name, algo_class in self._algorithms.items():
            algorithms.append({
                'name': algo_class.name,
                'description': algo_class.description,
                'category': algo_class.category,
                'type': 'builtin'
            })
        
        # 自定义算法
        for name, algo_class in self._custom_algorithms.items():
            algorithms.append({
                'name': algo_class.name,
                'description': algo_class.description,
                'category': algo_class.category,
                'type': 'custom'
            })
        
        return algorithms
    
    def get_algorithm(self, name: str) -> Optional[Type[BaseAlgorithm]]:
        """获取算法类"""
        if name in self._algorithms:
            return self._algorithms[name]
        if name in self._custom_algorithms:
            return self._custom_algorithms[name]
        return None
    
    def execute(self, algorithm_name: str, 
                input_data: AlgorithmInput,
                parameters: Dict[str, Any] = None) -> AlgorithmOutput:
        """
        执行算法
        
        Args:
            algorithm_name: 算法名称
            input_data: 输入数据
            parameters: 用户参数（覆盖默认值）
            
        Returns:
            算法输出
        """
        # 获取算法类
        algo_class = self.get_algorithm(algorithm_name)
        if algo_class is None:
            output = AlgorithmOutput()
            output.success = False
            output.message = f"算法 '{algorithm_name}' 不存在"
            return output
        
        # 创建算法实例
        algorithm = algo_class()
        
        # 合并参数
        if parameters:
            input_data.parameters = {**algorithm.get_default_parameters(), 
                                    **parameters}
        else:
            input_data.parameters = algorithm.get_default_parameters()
        
        # 验证输入
        valid, message = algorithm.validate_input(input_data)
        if not valid:
            output = AlgorithmOutput()
            output.success = False
            output.message = message
            return output
        
        # 执行算法
        try:
            output = algorithm.run(input_data)
            
            # 记录执行历史
            self._execution_history.append({
                'algorithm': algorithm_name,
                'parameters': input_data.parameters,
                'success': output.success,
                'timestamp': datetime.now()
            })
            
            return output
            
        except Exception as e:
            output = AlgorithmOutput()
            output.success = False
            output.message = f"算法执行错误: {str(e)}"
            return output
    
    def _test_algorithm(self, algorithm_class: Type[BaseAlgorithm]) -> Dict:
        """测试自定义算法"""
        try:
            # 创建测试实例
            algo = algorithm_class()
            
            # 检查必需属性
            assert hasattr(algo, 'name') and algo.name
            assert hasattr(algo, 'description')
            assert hasattr(algo, 'run')
            
            # 检查参数模式
            schema = algo.get_parameter_schema()
            assert isinstance(schema, dict)
            
            return {'valid': True, 'message': ''}
            
        except Exception as e:
            return {'valid': False, 'message': str(e)}
```
