## 1. 软件概述

NeuroPrime是一款面向神经科学研究的专业电生理数据分析平台，专注于猕猴脑电信号的处理、分析和可视化。软件采用模块化架构设计，支持灵活的算法扩展和自定义功能。

### 1.1 核心特点

- **专业性强**: 针对猕猴脑电生理数据优化，支持多种专业数据格式
- **模块化设计**: 算法、数据、UI分离，便于扩展和维护
- **可视化丰富**: 提供多种数据可视化方式，支持交互式分析
- **自定义算法**: 支持用户编写和导入自定义分析算法
- **工程化管理**: 完整的工程生命周期管理，数据组织清晰

### 1.2 适用场景

- 猕猴视觉神经科学实验数据分析
- 电生理信号处理与特征提取
- 神经元活动模式分析
- 刺激-响应关系研究
- 神经解码与分类研究

## 2. 运行环境

### 2.1 硬件要求

| 组件 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | Intel Core i5 | Intel Core i7 或更高 |
| 内存 | 8GB RAM | 16GB RAM 或更高 |
| 硬盘 | 100GB 可用空间 | 500GB SSD |
| 显卡 | 支持OpenGL | 独立显卡（用于可视化） |
| 显示器 | 1920x1080 | 2560x1440 或更高 |

### 2.2 软件环境

- **操作系统**: Windows 10/11 (64位), macOS 10.15+, Linux Ubuntu 18.04+
- **Python版本**: 3.8 - 3.11
- **主要依赖库**:
  - PyQt6 >= 6.4.0 (GUI框架)
  - NumPy >= 1.21.0 (数值计算)
  - SciPy >= 1.7.0 (科学计算)
  - Matplotlib >= 3.5.0 (数据可视化)
  - h5py >= 3.6.0 (HDF5文件操作)
  - scikit-learn >= 1.0.0 (机器学习)
  - MNE-Python >= 1.0.0 (神经信号处理)

### 2.3 安装步骤

1. 克隆或下载项目代码
2. 安装依赖: `pip install -r requirements.txt`
3. 运行软件: `python run.py` 或双击 `start.bat`

## 4. 具体功能

### 4.1 工程管理功能

#### 4.1.1 新建工程
- 创建新的分析工程，建立工程目录结构
- 自动生成project.json配置文件
- 工程结构包含：Document、Backup、data(raw/processed)等目录

#### 4.1.2 打开工程
- 打开已有工程，加载project.json配置
- 自动索引工程中的所有数据文件

#### 4.1.3 导入试验数据
- 导入配套数据文件（.ns3/.nev/.mbm）
- 自动解析并转换为HDF5格式存储
- 支持批量导入文件夹中的所有试验数据

### 4.2 数据浏览功能

#### 4.2.1 数据导航树
- 树形结构显示工程中的所有试验和数据项
- 试验列表可展开/折叠
- 数据类型分类：LFP数据、Spike数据、行为数据
- 双击数据项添加至分析列表

#### 4.2.2 数据预览
- 预览选中数据的基本信息和统计
- 显示采样率、通道数、时长
- 显示数据范围、均值、标准差

### 4.3 算法分析功能

#### 4.3.1 Spike检测与排序

**SpikeDetectionThreshold** (阈值检测)
- 基于电压阈值的Spike检测
- 可调参数：阈值系数、最小间隔时间
- 输出：Spike时间戳、波形、通道信息

**SpikeSortingPCA** (PCA排序)
- 基于PCA和K-Means的Spike排序
- 可调参数：PCA维度、聚类数量
- 输出：分类标签、特征空间可视化

#### 4.3.2 LFP信号分析

**LFPPowerSpectrum** (功率谱分析)
- 计算LFP功率谱密度
- 可调参数：频率范围、窗函数类型
- 输出：功率谱密度图

**LFPSpectrogram** (时频分析)
- 计算LFP时频谱图
- 可调参数：时间窗口、频率分辨率
- 输出：时频热力图

#### 4.3.3 行为数据分析

**PSTHAnalysis** (刺激后时间直方图)
- 分析刺激前后神经元发放模式
- 可调参数：时间窗口、分箱大小
- 输出：PSTH直方图

**RasterPlotAnalysis** (栅格图分析)
- 显示多个试次的Spike发放栅格图
- 可调参数：时间范围、试次筛选
- 输出：栅格图

**TuningCurveAnalysis** (调谐曲线分析)
- 分析神经元对不同刺激的调谐特性
- 可调参数：刺激条件、平滑参数
- 输出：调谐曲线图

**ROCAnalysis** (ROC曲线分析)
- 评估神经元区分不同刺激的能力
- 可调参数：正类标签、阈值范围
- 输出：ROC曲线、AUC值

#### 4.3.4 解码分析 (已注释，暂不启用)

**LDADecoder** (LDA解码)
- 使用线性判别分析进行刺激解码
- 可调参数：求解器、降维维度、交叉验证折数
- 输出：分类准确率、混淆矩阵

**SVMClassifier** (SVM分类)
- 使用支持向量机进行分类
- 可调参数：核函数、正则化参数、核系数
- 输出：分类准确率、支持向量信息

**RandomForestClassifier** (随机森林分类)
- 使用随机森林进行分类
- 可调参数：树的数量、最大深度、特征采样
- 输出：分类准确率、特征重要性

#### 4.3.5 MNE算法集成

集成MNE-Python库，支持以下算法：
- 坏通道检测 (MNEBadChannelDetection)
- 滤波 (MNEFiltering)
- 重采样 (MNEResampling)
- ICA分解 (MNEICA)
- ICA伪迹去除 (MNEICAArtifactRemoval)
- 功率谱 (MNEPowerSpectrum)
- 时频分析 (MNETimeFrequency)
- ERP分析 (MNEERP)
- 源定位 (MNESourceLocalization)
- 波束形成 (MNEBeamforming)
- 偶极子拟合 (MNEDipoleFitting)
- 功能连接 (MNEFunctionalConnectivity)
- 有效连接 (MNEEffectiveConnectivity)
- 频谱连接 (MNESpectralConnectivity)

### 4.4 自定义算法功能

#### 4.4.1 算法导入
- 支持导入Python编写的自定义算法
- 自动识别算法类和参数
- 参数配置界面自动生成

#### 4.4.2 算法管理
- 查看已导入的自定义算法
- 编辑算法代码
- 删除自定义算法

### 4.5 可视化功能

#### 4.5.1 数据可视化
- 原始信号波形显示
- 多通道信号叠加显示
- 时间对齐可视化

#### 4.5.2 分析结果可视化
- 直方图、曲线图、热力图
- 混淆矩阵显示
- 统计信息展示

#### 4.5.3 交互功能
- 缩放、平移、重置视图
- 保存图表为图片

### 4.6 数据导出功能

- 导出分析结果为HDF5格式
- 导出图表为PNG/PDF格式
- 导出统计数据为CSV格式

## 5. 使用指南

### 5.1 快速开始

1. **创建工程**
   - 点击"文件" → "新建工程"
   - 输入工程名称和路径
   - 点击"创建"

2. **导入数据**
   - 点击"数据加载"标签
   - 点击"导入数据"按钮
   - 选择.ns3/.nev/.mbm文件
   - 等待数据解析完成

3. **运行分析**
   - 在左侧导航树中选择数据
   - 双击添加到数据选择列表
   - 选择分析算法标签
   - 配置参数
   - 点击"运行分析"

4. **查看结果**
   - 在右侧可视化区域查看结果
   - 可保存图表或导出数据

### 5.2 数据组织

工程目录结构：
```
{project_name}/
├── project.json          # 工程配置
├── Document/             # 文档目录
├── Backup/               # 备份目录
├── src/                  # 源代码目录
└── data/                 # 数据目录
    ├── raw/              # 原始数据
    └── processed/        # 处理后数据(HDF5)
```

### 5.3 算法使用

#### 5.3.1 内置算法
1. 选择对应的算法标签页
2. 在左侧导航树双击选择数据
3. 在参数面板配置参数
4. 点击"运行分析"

#### 5.3.2 自定义算法
1. 点击"自定义算法"标签
2. 点击"导入算法"
3. 选择Python算法文件
4. 系统自动生成参数界面
5. 配置参数并运行

### 5.4 时间对齐

当选择多个试验的数据时：
1. 点击"时间对齐"按钮
2. 选择对齐方式（试次开始/刺激 onset）
3. 设置时间偏移
4. 确认对齐配置

### 6.2 变更的需求

#### 6.2.1 智能分析标签页
**原始需求**: 包含LDA解码、SVM分类、随机森林等解码分析算法

**变更说明**: 
- 已实现LDADecoder、SVMClassifier、RandomForestClassifier算法
- 但由于数据需求（需要多类别刺激条件），该标签页已注释掉，暂不显示
- 算法代码保留在src/algorithms/decoding.py中，可随时启用

**变更原因**:
- 解码分析需要数据包含多个刺激条件（stim_cnd）
- 当前测试数据大多只有一个刺激条件
- 为避免用户困惑，暂时隐藏该功能

#### 6.2.2 MNE算法集成
**原始需求**: 无

**新增功能**:
- 集成MNE-Python库
- 支持14种MNE算法
- 提供算法模板和导入机制

**新增原因**:
- 扩展软件功能，支持更多神经信号处理方法
- 利用MNE-Python强大的EEG/MEG分析能力

#### 6.2.3 数据需求提示
**原始需求**: 无

**新增功能**:
- 根据算法动态显示数据需求提示
- 数据类型不匹配时弹框提示

**新增原因**:
- 提升用户体验，减少因数据选择错误导致的问题
- 帮助用户理解不同算法的数据需求
